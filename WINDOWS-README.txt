Windows 사용자를 위한 빠른 가이드
═══════════════════════════════════

📁 배치 파일 (3개만!)

1. start.bat
   - 서버 시작
   - 브라우저 수동 실행
   
2. start-auto.bat  ⭐ 권장!
   - 서버 시작
   - 브라우저 자동 실행
   
3. stop.bat
   - 서버 종료


🚀 사용 방법

가장 쉬운 방법:
  1) start-auto.bat 더블클릭
  2) 끝! (브라우저 자동 열림)
  
종료:
  1) stop.bat 더블클릭


📖 더 자세한 내용

- WINDOWS-GUIDE.md 참조
- 문제 발생 시: WINDOWS-GUIDE.md의 "문제 해결" 섹션 참조
